# Technology Stack

## Programming Languages
- **PHP**: Server-side scripting for dynamic content and backend logic
- **HTML5**: Semantic markup with modern HTML standards
- **CSS3**: Styling with custom CSS and framework integration
- **JavaScript**: Client-side interactivity and dynamic behavior

## Frontend Framework & Libraries
- **Bootstrap 5.3.8**: Responsive CSS framework for layout and components
  - CDN: `https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css`
  - JS Bundle: `https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.bundle.min.js`
- **Font Awesome 7.0.1**: Icon library for UI elements
  - CDN: `https://cdnjs.cloudflare.com/ajax/libs/font-awesome/7.0.1/css/all.min.css`

## Development Environment
- **XAMPP**: Local development server stack
- **Apache**: Web server for PHP execution
- **File Structure**: Standard web application layout with assets separation

## Asset Management
- **CSS**: Custom stylesheets in `assets/css/style.css`
- **JavaScript**: Custom scripts in `assets/js/script.js`
- **Images**: Organized in `assets/images/` with web-optimized formats (JPEG, WebP)

## Browser Compatibility
- **Responsive Design**: Mobile-first approach with Bootstrap grid system
- **Modern Browser Support**: HTML5 and CSS3 features
- **Cross-Platform**: Compatible with desktop and mobile devices

## Development Commands
Since this is a PHP-based web application running on XAMPP:

1. **Start XAMPP**: Launch Apache server from XAMPP control panel
2. **Access Application**: Navigate to `http://localhost/phpcode/deer/`
3. **Admin Access**: Navigate to `http://localhost/phpcode/deer/admin_area/`
4. **File Editing**: Direct file modification in the project directory

## Dependencies
- **External CDNs**: Bootstrap and Font Awesome loaded via CDN
- **Local Assets**: Custom CSS, JavaScript, and images served locally
- **No Package Manager**: Direct file inclusion without npm/composer dependencies
